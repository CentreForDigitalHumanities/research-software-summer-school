# Working with the UNIX Shell (Carpentries)
If you are not yet familiar or comfortable with working in the command-line terminal using the shell, we recommend completing at least the first three modules of this Carpentries workshop: [https://preview.carpentries.org/lc-shell/index.html](https://preview.carpentries.org/lc-shell/index.html). 

You can complete this tutorial at your own pace, but make sure to be confident navigating the shell and executing files and programs from it before starting the summer school. Good luck and enjoy!
